用戶手冊

歡迎使用「慳真D」！本系統每日自動同步政府「網上價格一覽通」數據，助你輕鬆對比全港各大連鎖超市的最新價格。

🔍 核心功能與搜尋技巧

  • 對話式快捷搜尋： 於首頁輸入「品牌」或「貨品名稱」，即時為你展示官方數據中嘅超市定價及折扣詳情。
  • 一鍵篩選優惠： 善用首頁的「最新劈價精選」按鈕，過濾並只顯示有促銷活動的貨品。
  • 🔥 捕捉超大差價： 點擊 「🔥 今日超大差價」 快捷鍵，系統會幫你找出商戶間價格相差超過 10% 的貨品。
  • 智能換算單價： 系統會自動為你計算「買兩件」、「第二件半價」等複雜優惠的折合單價，並標示 💰。
  • 文字式快速導航： 想返主頁揀其他分類？直接喺對話框打「返主目錄」或「主頁」即可。

📸 AI 視覺辨識搜尋

影張相，AI 專業大腦會即刻幫你辨識產品並自動填入搜尋框。只要輸入 Gemini API Key，準確度即刻提升！

註：如未輸入 Key，系統會自動退守使用輕量級離線辨識模式。

🔔 智能到價提示功能

見到心水貨品想等減價？點擊產品卡片上的 橙色鐘仔 🔔，輸入你的目標價即可追蹤。當貨品跌穿目標價，畫面頂部的 🔔 就會亮起紅點！

🛡️ 免責聲明及私隱保障

1. 私隱安全： 本 App 不設任何個人帳戶系統，所有「到價提示」及「搜尋紀錄」均僅儲存於閣下的手機瀏覽器內。我們無法取得、監控或售賣您的個人數據。

2. 數據來源： 價格數據均取自香港政府「資料一線通 (Data.gov.hk)」的「網上價格一覽通」數據集，版權歸消費者委員會及相關商戶所有。

3. 免責聲明： 本 App 為開源免費工具，資訊僅供參考。超市實際價格及優惠可能隨時變更，購買前請以超市現場標籤為準。對於因資訊誤差引致的損失，本開發團隊概不負責。

📱 如何將 App 加入手機主畫面

Android： 用 Chrome 開啟 ➔ 點擊右上角「⋮」 ➔ 選擇「加至主畫面」。
iOS： 用 Safari 開啟 ➔ 點擊底部「分享」圖示 ➔ 選擇「加入主畫面」。
