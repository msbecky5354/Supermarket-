用戶手冊 User Manual

🇭🇰 繁體中文版

歡迎使用「香港生活百貨價格神器」！本系統每日自動同步政府「網上價格一覽通」數據，助你輕鬆比較全港各大超市及連鎖商戶（包含個人護理店等）的價格。

核心功能與搜尋技巧：

  • 智能對話式搜尋： 點選主分類後，可直接輸入「品牌」或「貨品名稱」尋找貨品。
  • 一鍵篩選優惠： 善用上方「淨睇優惠」快捷鍵，只顯示有促銷活動的貨品。
  • 智能換算單價： 自動計算買一送一等複雜優惠的折合單價。

如何加入手機主螢幕 (PWA)：

Android： 用 Chrome 開啟 ➔ 點擊右上角「⋮」 ➔ 加至主畫面。
iOS (iPhone)： 用 Safari 開啟 ➔ 點擊底部「分享」圖示 ➔ 加入主畫面。

🇨🇳 简体中文版

欢迎使用“香港生活百货价格神器”！本系统每日自动同步政府“网上价格一览通”数据，助您轻松比较全港各大超市及连锁商户的商品价格。

核心功能与搜索技巧：

  • 智能对话式搜索： 点击主分类后，可直接输入“品牌”或“商品名称”寻找商品。
  • 一键筛选优惠： 善用上方“只看优惠”快捷键，仅显示有促销活动的商品。
  • 智能换算单价： 自动计算买一送一等复杂优惠的折合单价。

如何加入手机主屏幕 (PWA)：

Android： 用 Chrome 打开 ➔ 点击右上角“⋮” ➔ 添加到主屏幕。
iOS (iPhone)： 用 Safari 打开 ➔ 点击底部“分享”图标 ➔ 添加到主屏幕。

🇬🇧 English Version

Welcome to the HK Smart Price Tracker! Our system automatically syncs daily with the government's data to help you compare prices across major retail chains.

Core Features & Search Tips:

  • Smart Search: Type a brand or product name in the chat box to find items.
  • Discount Filter: Use the "Discounted Only" chip to show products on sale.
  • Unit Price Calculation: Automatically calculates the average price for promotions like Buy 1 Get 1 Free.

How to Add to Home Screen:

Android: Open in Chrome ➔ Tap top-right menu "⋮" ➔ Add to Home screen.
iOS (iPhone): Open in Safari ➔ Tap bottom "Share" icon ➔ Add to Home Screen.
